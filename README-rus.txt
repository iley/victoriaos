VictoriaOS v 0.03 ReadMe

-------------------------------------------------------------------------------
ВВЕДЕНИЕ

VictoriaOS - это 16-битная операционная система, созданная в образовательных 
целях. Эта система в ее сегодняшнем виде не претендует ни на какое 
соперничество с другими современными системами. Она была создана исключительно 
в целях изучения общих принципов создания операционных систем и испытания своих 
идей. 

Таким образом, нынешнюю версию VictoriaOS можно считать своеобразной "пробой 
пера" и демонстрацией способностей. Следующая версия этой системы, возможно, 
будет написана с нуля, и она не будет иметь ничего общего с нынешней (кроме, 
конечно, названия). 

Создателям хотелось бы, чтобы вокруг этого небольшого проекта собирались люди, 
которым интересна разработка операционных систем. Быть может, со временем из 
нынешней примитивной Виктории вырастет серьезная операционная система. 

-------------------------------------------------------------------------------
СИСТЕМНЫЕ ТРЕБОВАНИЯ

VictoriaOS работает в реальном режиме процессора, ее ядро использует только 
возможности процессора Intel 80186. Разумеется, она может работать на любом
процессоре, совместимом с Intel 80186 (подавляющее большинство современных 
процессоров).

VictoriaOS загружается с одной дискеты формата 3.5''

-------------------------------------------------------------------------------
КОМПИЛЯЦИЯ И ЗАПУСК

Компиляция VictoriaOS может быть выполнена в операционных системах Linux
и Windows. Компиляция выполняется одинаковым образом в этих системах.

Чтобы скомпилировать VictoriaOS требуется Flat Assembler (заведомо подходит
версия 1.67) и утилита make. Также испольуется утилита imgwrite, которая 
поставляется вместе с VictoriaOS в виде исполняемых файлов для Windows и Linux,
а также исходных кодов на языке С. Также с VictoriaOS поставляется Flat 
Assember (см. лицензию в файле fasm/fasm-license.txt) для Windows и Linux и 
утилиты make и rm для Windows (в виде исполняемых файлов).

Чтобы скомпилировать VictoriaOS, распакуйте архив с исходными кодами в любую
доступную директорию на вашем компьютере, сделайте директорию с исходными 
кодами текущей и выполните команду "make".

После компиляции будет создан файл disk.img. Его можно записать на дискету
или использовать с виртуальной машиной (например qemu, vmware или MS Virtual
PC).
